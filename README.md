# [WA-Delivery](https://github.com/dudushy/WA-Delivery/)
A WhatsApp BOT to send custom messages to a list of contacts.

---

## Prerequisites:
- [NodeJS](https://nodejs.org/)
- [Google Chrome](https://www.google.com/chrome/)

## How to install:
```bash
npm i
```
## How to run:


```bash
npm start
```
